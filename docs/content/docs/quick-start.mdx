# Get Started in 5 Minutes

Upload files to S3 with **zero configuration** in just 3 steps.

## 1. Install

```bash
npm install -g create-next-s3-uploader
```

## 2. Initialize Your Project

```bash
npx create-next-s3-uploader init
```

The CLI will:

- âœ… Set up your API route (`/api/s3-upload`)
- âœ… Configure your storage provider (AWS S3, Cloudflare R2, etc.)
- âœ… Generate type-safe client code
- âœ… Create example components

## 3. Start Uploading

```typescript
// In your component
import { upload } from "@/lib/upload-client";

export function ImageUpload() {
  const { uploadFiles, files, isUploading } = upload.imageUpload();

  return (
    <div>
      <input
        type="file"
        accept="image/*"
        onChange={(e) => uploadFiles(e.target.files)}
      />

      {files.map((file) => (
        <div key={file.id}>
          <span>{file.name}</span>
          <progress value={file.progress} max={100} />
          {file.url && <img src={file.url} alt={file.name} />}
        </div>
      ))}
    </div>
  );
}
```
