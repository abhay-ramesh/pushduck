# Next S3 Uploader

**Upload files to S3 in 5 minutes** with full TypeScript support and zero configuration.

## ✨ What You Get

- **🚀 5-Minute Setup**: CLI handles all the configuration
- **🔒 Type Safe**: Full TypeScript inference from server to client
- **🌐 Any Provider**: AWS S3, Cloudflare R2, DigitalOcean, MinIO, Google Cloud
- **📱 Modern UI**: Drag & drop, progress tracking, beautiful components
- **⚡ Zero Config**: Just run the CLI and start uploading

## 🎯 Quick Example

```typescript
// 1. Install & initialize
npx create-next-s3-uploader init

// 2. Use in your component
const { uploadFiles, files, isUploading } = upload.imageUpload();

// 3. Upload files
await uploadFiles(selectedFiles);
```

That's it! You're now uploading files to the cloud with full type safety.

## 🌟 Why Choose Next S3 Uploader?

### **No Configuration Headaches**

The CLI sets up everything automatically - routes, types, client configuration. Just focus on building your UI.

### **Full Type Safety**

```typescript
// Property-based access with full inference
const { uploadFiles } = upload.imageUpload(); // ✅ Type-safe
// No more string literals and typos

```

### **Any Storage Provider**

Works with AWS S3, Cloudflare R2, DigitalOcean Spaces, MinIO, Google Cloud Storage, and more.

### **Production Ready**

Built-in error handling, progress tracking, file validation, and performance optimization.
